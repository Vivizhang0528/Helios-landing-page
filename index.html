PRIMARY Helios

Heliopause · harness boundary

Where the solar wind meets the void

In astrophysics, the heliopause is the boundary where the Sun’s influence gives way to interstellar space. In Ship, that boundary is your harness: phases, gates, and evidence—not vibes. Helios is the coordinating core; each world-sized agent runs a bounded arc. Nothing crosses the heliopause without a verdict.

GitHub Pipeline

agents

Single-thread chat collapses into self-review and drift. Ship uses separate contexts—each agent a small world with its own orbit—so implementation, review, and QA cannot pretend they are the same voice.

Self-certification

When builder and judge share one model session, defects skew invisible.

Narrative “done”

Without independent QA against a running system, acceptance stays rhetorical.

Process entropy

Without on-disk state and hooks, long tasks melt back into unstructured chat.

Pipeline

Like CI stages: design → dev → review → qa → simplify → handoff. Each transition is a gate, not a suggestion.

DESIGN · SKILL

Vesta

A differentiated protoplanet—layered, structured crust. In Ship, the design skill does the same for your repo: adversarial investigation, a single merged spec, and an execution drill so stories are implementable before dev spends tokens.

DEV · SKILL

Pallas

Irregular, dense, mineral-bright. dev is where bulk lands: story-by-story implementation, wave scheduling when worktrees are available, and merge-back rules so parallel streams don’t fork reality.

SETUP · SKILL

Gaspra

First close-imaged small body—raw surface, real instruments. setup is your first flyby: stack detection, CI / pre-commit, AGENTS.md and .learnings, plus a harness audit so the pipeline matches the repo.

REVIEW · SKILL

Eros

Close-approach geometry—every facet lit differently. review is staff-level diff scrutiny: P1/P2/P3 findings, evidence-backed claims, and one diagnosis when multiple bugs share a root cause.

REFACTOR · SKILL

Ida

Binary system—primary and satellite locked in the same gravity budget. refactor makes coupling explicit: surgical edits in-file or structural moves across modules without changing behavior—stop-gates still apply.

QA · SKILL

Ryugu

Dark rubble pile—truth comes from touch, not from slides. qa is independence-bound: it checks the running product against MUST / SHOULD criteria with L1–L3-style evidence, not the review narrative.

HANDOFF · SKILL

Bennu

Sample return—payload has to survive atmosphere and land in the right lab. handoff shepherds the PR, CI failures, review threads, and merge conflicts until checks are green or retries exhaust—evidence over “LGTM”.
